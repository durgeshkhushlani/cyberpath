# CyberPath | Master the Art of Security 🛡️

CyberPath is a dynamic, full-stack cybersecurity career roadmap platform designed to track, guide, and accelerate the learning journey of aspiring InfoSec professionals. Whether you are aiming for a slow and foundational deep-dive or a rapid intensive bootcamp, CyberPath maps your daily tasks, resources, and progress securely in one sleek interface.

## 🚀 Key Features

* **Dynamic Curriculum Engine:** Choose between Deep Dive (8 weeks), Standard (4 weeks), or Intensive (2 weeks) pathways. 
* **Intuitive Progress Tracking:** Visualize your learning streak manually or toggle between task-based and day-based viewing modes to fit your personal studying preference.
* **Modern UI/UX:** Built with a fully responsive architecture, mobile drawer navigation, and seamless global Light/Dark mode toggling.
* **Integrated Tools & Resources:** Instant access to curated labs, external study hubs, and actionable daily toolkit tips directly from your dashboard.
* **Secure Auth & Onboarding:** JSON Web Token (JWT) secured profile management.

## 🛠️ Technology Stack

* **Frontend:** React.js (Vite), Tailwind CSS, Zustand (State Management), Lucide Icons
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Deployment:** Vercel (Frontend) & Render (Backend API)
